🏨 Booking Manager

Connect to your bookings

This page is public but holds no data — it reads your private booking-manager repo with a token stored only in this browser.

Create one at github.com/settings/personal-access-tokens/new:

  1. Repository access: Only select repositories → booking-manager
  2. Permissions → Contents: Read and write (write lets you add bookings from here)
  3. Permissions → Actions: Read and write (lets you trigger checks)

＋ Add a booking

Paste your booking confirmation email and hit Parse. The hotel's page URL and the first price check happen automatically — run a check after saving, or wait for the next scheduled one.
